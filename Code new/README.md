# Graphiti Tutorial: From Beginner to Advanced

Welcome to the Graphiti tutorial! This tutorial is designed to help you learn the ins and outs of the Graphiti library, whether you're just starting out or looking to build complex AI applications.

## About This Tutorial

This tutorial is divided into two parts:

-   **Beginner**: Start here if you're new to Graphiti. You'll learn the fundamental concepts of knowledge graphs, how to add data, and how to perform basic queries.
-   **Advanced**: Once you've mastered the basics, move on to the advanced tutorial to learn how to build a fully-fledged conversational AI agent using Graphiti and LangGraph.

## How to Get Started

1.  **Navigate to the `beginner` directory**:
    ```bash
    cd beginner
    ```
2.  **Follow the instructions in the `README.md` file there** to set up your environment and run the beginner's tutorial.
3.  **Once you're comfortable, move on to the `advanced` directory** and follow the instructions in its `README.md`.

We hope you enjoy learning about Graphiti!
